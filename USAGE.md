# Инструкция по использованию

## Что показывает приложение

На странице отображается гриф гитары:
- слева `0` лад (нулевой),
- справа последние лады (по умолчанию `24`),
- ширина ладов пропорциональна реальному грифу,
- сверху есть маркеры ладов,
- ноты выбранной гармонии/аккорда подсвечиваются разными цветами по степеням.

## Основные элементы управления

## 1) Root (тональность)

Выберите основную ноту (`C`, `C#`, `D` и т.д.) — от неё считаются гаммы и аккорды.

## 2) Режим

- `Гармония` — подсветка гаммы/лада по всему грифу или в позиции.
- `Аккорд` — отображение готовых аппликатур (voicing) для выбранного типа аккорда.

## 3) Гармония (в режиме `Гармония`)

- `Семейство`: пентатоника / диатоника / блюз / джазовые
- `Лад / гамма`: конкретная гамма или лад внутри выбранного семейства

После выбора на грифе подсвечиваются ноты этой гармонии.

## 4) Аккорд (в режиме `Аккорд`)

- `Тип аккорда`: `Maj`, `Min`, `7`, `Maj7`, `m7`, и т.д.
- `Аппликатура`: конкретный voicing (если доступно несколько)

Если для выбранной позиции нет готовой аппликатуры, будет показано сообщение.

## 5) Отображение

- `Весь гриф` — все лады
- `Позиция` — окно позиции (с CAGED-меткой)

Дополнительно:
- `Позиция` — номер позиции
- `Форма (CAGED)` — фильтр по форме (`C/A/G/E/D`)
- `Вне позиции` — скрывать или приглушать ноты вне окна

## 6) Строй

### Пресеты

Выберите пресет (`Standard`, `Drop`, `Open`) и нажмите `Применить пресет`.

### Свой строй (custom)

Введите 6 нот (от 6-й струны к 1-й) и нажмите `Применить свой строй`.

Допускаются:
- диезы (`#`) — например `F#`
- бемоли (`b`) — например `Bb` (будут автоматически нормализованы)

## 7) Опции

- `Показывать подписи` — подписи нот/степеней на точках
- `Показывать открытые струны` — включает/выключает открытые струны
- `Сбросить настройки` — возврат к дефолтному состоянию

## Цвета на грифе

Цвета соответствуют степеням (интервалам от root), например:
- `1` (тоника)
- `b3 / 3`
- `5`
- `b7 / 7`

Тоника выделяется сильнее (размером/обводкой).

## Сохранение настроек

Приложение сохраняет выбранные настройки в `localStorage`, поэтому после перезагрузки страницы состояние восстанавливается.
