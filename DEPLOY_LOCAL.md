# Локальный запуск (через `run.sh`)

Этот проект является статическим сайтом (`HTML/CSS/JS`) и не требует сборки.

## Требования

- `bash`
- `python3` (используется встроенный `http.server`)

## Запуск

В корне проекта:

```bash
./run.sh
```

Скрипт:
- находит свободный локальный порт,
- запускает HTTP-сервер на `127.0.0.1`,
- сохраняет `PID` и `PORT` в служебные файлы:
  - `.server.pid`
  - `.server.port`
  - лог: `.server.log`

После запуска в консоли появится URL вида:

```text
Started: http://127.0.0.1:48561 (pid=12345)
```

Откройте его в браузере.

## Остановка

```bash
./stop.sh
```

Скрипт прочитает `.server.pid`, остановит процесс сервера и удалит служебные файлы.

## Типичные проблемы

## `PermissionError` при запуске

Если запуск происходит в ограниченной песочнице/CI-среде, создание сокета может быть запрещено. На обычной локальной машине `run.sh` работает нормально.

## Порт не отвечает

- Проверьте `.server.log`
- Перезапустите:
  ```bash
  ./stop.sh
  ./run.sh
  ```

## Альтернативный запуск (без скриптов)

Если нужно вручную:

```bash
python3 -m http.server 8000 --bind 127.0.0.1 --directory .
```
